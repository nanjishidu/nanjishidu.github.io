ubuntu14.04安装初始化

June 4, 2014 in technology

记录ubuntu系统初始化安装过程。

1.

sudo apt-get update
sudo apt-get upgrade

  1. chrome

    sudo apt-get install chromium-browser
sudo apt-get install pepperflashplugin-nonfree
sudo update-pepperflashplugin-nonfree --install

卸载命令：

sudo apt-get remove chromium-browser pepperflashplugin-nonfree安装

  1. vim

    sudo apt-get install vim

  2. golang环境变量

    export GOROOT=/opt/environment/go
export GOBIN=$GOROOT/bin
export GOPATH=/opt/environment/work
export LITEPATH=/opt/environment/liteide/bin
export BINPATH=/opt/bin
export PATH=.:$GOBIN:$LITEPATH:$BINPATH:$PATH

  3. 安装wine1.7

    sudo add-apt-repository ppa:ubuntu-wine/ppa
sudo apt-get update && sudo apt-get install wine1.7

    6.安装 debain 2013 qq

    http://www.longene.org/download/

    启动不了就安装它吧

    sudo apt-get install libgtk2.0-0:i386

  4. ubuntu已经废弃chkconfig使用 sysv-rc-conf代替

  5. 安装postgresql

    sudo apt-get install postgresql 
sudo passwd postgres //修改postgres密码
su postgres 进入postgres用户
psql 后	
ALTER USER postgres WITH PASSWORD '111111';修改数据库密码
\q   退出

  6. dpkg -l 列表已安装软件包

    dpkg -r  删除

  7. git

    sudo apt-get install git

  8. mercurial

    sudo apt-get install mercurial

  9. geany

    sudo apt-get install geany

  10. apache2

    sudo apt-get install apache2
sudo /etc/init.d/apache2 restart

  11. php5

    sudo apt-get install php5
sudo apt-get install  php5-mysql php5-gd php5-mcrypt php5-xdebug php5-snmp php5-sqlite php5-rrd php5-pgsql php5-memcache php5-gd

  12. mysql-server

    sudo apt-get install mysql-server

  13. 安装Svn客户端

    sudo add-apt-repository ppa:rabbitvcs/ppa
sudo apt-get update
sudo apt-get install rabbitvcs-core rabbitvcs-nautilus3 rabbitvcs-gedit rabbitvcs-cli
sudo apt-get install subversion

  14. 输入法

    sudo apt-get install fcitx-table-wbpy
