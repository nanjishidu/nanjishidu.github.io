Mac使用过程小技巧

August 1, 2015 in technology

mac下清除文件扩展属性,清除系统睡眠文件。

1.文件扩展属性带有@

有@表示该文件或目录带有扩展属性。有时候用 Time Machine 备份过的文件会出现这种现象。使用以下命令清除：

xattr -c

2.清除系统睡眠文件

实际使用过程中很少关机。系统的睡眠文件会多达4G或者更多。 删除休眠文件

sudo rm -f  /var/vm/sleepimage

关闭休眠

sudo pmset hibernatemode 0

开启休眠

sudo pmset hibernatemode 0
