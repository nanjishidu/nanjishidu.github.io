树莓派时间同步

March 17, 2017 in technology

树莓派没有硬件时钟，重启后时间就会重置，通过ntpdate命令进行时间同步。

树莓派上运行btsync，有的时候会提示时间误差。正常情况下重启服务就可以。

但重启了几次后问题依然存在，看了下系统时间发现时间不正确。可能与最近服务断电重启有关。

通过ntp可以修改系统时间，但是一直没有亲自操作过。

安装ntp后执行时间同步,发现时间有8个小时的误差,修改时区后正常。特记录

1.同步网络时间

sudo apt-get install ntpdate
ntpdate 0.pool.ntp.org

2.修改/etc/localtime解决时区问题

cp /etc/localtime /etc/localtime.bak
cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
date

3.通过tzselect修改时区

tzselect
