npm-check-updates升级element-ui2.x

December 9, 2017 in technology

element-ui发布2.x版本后，1.x版本停止维护。使用npm-check-updates检测项目package.json升级第三方包。

一、npm-check-updates

  • 安装npm-check-updates，(alias)ncu

    npm install -g npm-check-updates

  • 升级项目中所有升级包

    ncu -a

二、element-ui升级1.x到2.x版本

  • ncu升级element-ui2.x版本

    ncu -u element-ui

  • 修改2.x主题

    在main.js中修改
import 'element-ui/lib/theme-default/index.css'
为
import 'element-ui/lib/theme-chalk/index.css'
